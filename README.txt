Hi, this is README.

Environment: Mac OS 10.13.6  Ruby 2.3.7

This is an implement for a C translator, it follows a LL(1) grammar and translator a C code into assembly like code. The file scanner.rb is the scanner for the translator it takes C code and output a bunch of tokens. The translator takes these tokens and translator into assembly like code. It contains a recursive descent parser. The folder test files has some test files to test the translator.
